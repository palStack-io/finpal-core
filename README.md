## The Story Behind finPal

**finPal started as Dollar Dollar Bill Y'all** - and it started with roommates arguing over who paid for pizza.

You know the drill: someone buys groceries, someone else pays rent, a third person covers utilities. By month's end, nobody knows who owes what. Spreadsheets get messy. Group chats fill with "did I pay you back?" messages. Friendships get awkward over $20.

One of us got tired of it and built **Dollar Dollar Bill Y'all**. Simple goal: track who owes whom, settle up, move on with life. No bank connections, no subscriptions, no data harvesting. Just math and fairness.

**Then it worked. Really well.**

Friends started using it. Then those friends' friends. People we'd never met were running it on their servers to split bills with their roommates.

And the feature requests kept coming:
- "Can it track my budget too?"
- "What about my investment portfolio?"
- "SimpleFin for automatic bank sync?"
- "Can we use this for our monthly dinner club expenses?"

Each request made sense. Each solved a real problem. So we kept building.

The codebase grew. The vision expanded. What started as roommate bill-splitting became a full financial management platform. We realized we'd outgrown both the name and the original architecture.

**Dollar Dollar Bill Y'all evolved into finPal.**

Complete rewrite. Modern stack. All the features people kept asking for: budgeting, investment tracking, bank sync, enhanced bill-splitting. We kept what worked (privacy-first, self-hosted, no tracking) and built everything else around it.

**But finPal isn't the endgame‚Äîit's the foundation.**

Our ultimate goal is **debtFree**: a gamified debt payoff platform with financial literacy at its core. Think progress bars that actually motivate you. Achievements for paying off credit cards. Educational content that doesn't feel like homework. Community challenges that make debt reduction social and supportive.

finPal is teaching us what people need for financial management. debtFree will take those lessons and laser-focus them on the debt crisis millions face. Financial literacy shouldn't be locked behind paywalls or boring textbooks‚Äîit should be built into the tools you use daily.

**The Roadmap:**
```
Dollar Dollar Bill Y'all (2023) - Roommate bill-splitting
    ‚Üì
finPal (2024-2025) - Complete financial management
    ‚Üì
debtFree (Future) - Gamified debt payoff + financial literacy
```

finPal is where we learn. debtFree is where we focus that knowledge on helping people escape debt.

**The Privacy Stand**
Every mainstream money app wants your bank credentials, tracks every purchase, and monetizes your spending patterns. We built the opposite: a financial platform that respects your privacy because it literally can't access what it doesn't collect.

**Learning Through Building**
None of us came from fintech backgrounds. We learned secure financial data handling, bank APIs, multi-currency systems, and compliance stuff as we built. AI assistants (Claude, ChatGPT, Qwen) helped us learn faster, but every line gets human review.

finPal is now part of **palStack**‚Äîour collection of privacy-first tools. From roommate debt tracking to complete financial management, all without selling your data.

**Legacy Note:** The [original Dollar Dollar Bill Y'all repository](https://github.com/harung1993/dollardollar) is still there for anyone who just needs the simpler bill-splitting focus.

## Quick Start

### Deploy with GitHub Container Registry (Recommended)

Pull pre-built images from GitHub Container Registry:

```bash
# Clone repository
git clone https://github.com/palStack-io/finpal-core.git
cd finpal-core

# Configure environment
cp .env.example .env
# Edit .env with your settings (see Configuration section)

# Start finPal
docker-compose up -d

# Access at http://localhost:5000
```

**First Time Setup:**
1. Open http://localhost:5000 in your browser
2. Create your account (first user becomes admin)
3. Configure your default currency in Settings
4. (Optional) Set up SimpleFin for bank sync
5. Start tracking your finances!

**üìö For comprehensive installation, configuration, and deployment guides, visit:**
**[finpal.palstack.io/docs](https://finpal.palstack.io/docs)**

- Docker and Docker Compose installed

- 2GB RAM minimum (4GB recommended)
- 10GB disk space
- (Optional) SimpleFin account for bank sync (US banks)
- (Optional) SMTP server for email notifications
- (Optional) OAuth2 provider for OIDC/SSO

**üìö Detailed setup guides available at [finpal.palstack.io/docs](https://finpal.palstack.io/docs)**

## Configuration

### Environment Variables

finPal can be configured via environment variables in your `.env` file:

#### Core Settings
| Variable | Default | Description |
|----------|---------|-------------|
| `DB_USER` | `finpal` | PostgreSQL username |
| `DB_PASSWORD` | - | PostgreSQL password |
| `DB_NAME` | `finpal` | Database name |
| `SECRET_KEY` | - | Flask secret key |
| `JWT_SECRET_KEY` | - | JWT signing key |
| `DEFAULT_CURRENCY` | `USD` | Default currency code |

#### Features
| Variable | Default | Description |
|----------|---------|-------------|
| `INVESTMENT_TRACKING_ENABLED` | `false` | Enable portfolio features |
| `SIMPLEFIN_ENABLED` | `false` | Enable SimpleFin bank sync |

#### Authentication
| Variable | Default | Description |
|----------|---------|-------------|
| `OIDC_ENABLED` | `false` | Enable OIDC/SSO authentication |
| `OIDC_CLIENT_ID` | - | OAuth2 client ID |
| `OIDC_CLIENT_SECRET` | - | OAuth2 client secret |
| `OIDC_PROVIDER_NAME` | `OIDC` | Display name for provider |
| `OIDC_DISCOVERY_URL` | - | OIDC discovery endpoint |

### Example Configuration

```bash
# .env file

# Database
DB_USER=finpal
DB_PASSWORD=your_secure_password
DB_NAME=finpal

# Security
SECRET_KEY=your_secret_key_here
JWT_SECRET_KEY=your_jwt_secret_here

# Application
DEFAULT_CURRENCY=USD

# Features
INVESTMENT_TRACKING_ENABLED=true
SIMPLEFIN_ENABLED=true

# Optional: Google OAuth
OIDC_ENABLED=true
OIDC_CLIENT_ID=your-google-client-id
OIDC_CLIENT_SECRET=your-google-client-secret
OIDC_DISCOVERY_URL=https://accounts.google.com/.well-known/openid-configuration
OIDC_PROVIDER_NAME=Google
```

## SimpleFin Bank Sync Setup

Connect your US bank accounts automatically:

1. **Get SimpleFin Account**
   - Visit [simplefin.org](https://simplefin.org)
   - Create account and connect your bank
   - Generate a setup token

2. **Configure in finPal**
   - Go to Settings ‚Üí Bank Connections
   - Paste the setup token
   - Select which accounts to sync
   - Transactions import automatically

**Privacy Note:** SimpleFin acts as a bridge to your bank. finPal stores only the access URL locally‚Äîyour bank credentials never touch finPal.

## Why Self-Hosted?

Because your financial data is nobody's business but yours. You shouldn't need:
- Permission from a cloud service to see your own transactions
- A subscription to track your own money
- To trust a third party with your complete financial profile
- Internet connectivity to check your budget

Self-hosting means:
- Complete privacy and control
- No recurring costs
- Works offline
- Integrate with anything
- Modify as needed
- Your data never leaves your server
