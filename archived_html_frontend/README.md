# Archived HTML Frontend

**Date Archived:** December 25, 2024

## What's Here

This folder contains the original Flask HTML template-based frontend for Dollar Dollar Bill Y'all.

### Contents:
- `templates/` - Jinja2 HTML templates
- `static/` - CSS, JavaScript, and images
  - `static/css/` - Stylesheets (styles.css, investment-charts.css, etc.)
  - `static/images/` - Logos (dddby.png, dollar.png)
  - `static/js/` - Frontend JavaScript

## Design Theme (Preserved)

The original theme used these beautiful colors:
- **Primary Green:** #15803d
- **Dark Green:** #166534
- **Accent Gold:** #fbbf24
- **Light Gold:** #fef3c7
- **Dark Backgrounds:** #111827, #030712

## Why Archived?

We're migrating to a modern React-based frontend with:
- **Web App:** Vite + React (dollardollar-web/)
- **Mobile App:** React Native/Expo (dollardollar-mobile/)
- **Branding:** PalStacks umbrella with DollarPal/EuroPal naming

The theme, colors, and design philosophy are being ported to the new React apps.

## Original Branding

- App Name: "Dollar Dollar Bill Y'all"
- Logo: dddby.png
- Style: Dark theme with money symbol grid background
